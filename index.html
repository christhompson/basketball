basketball by christhompson

A simple joke "choose your own adventure" game (where you don't actually have any choice)

This is a quick "game", as some might call it, that I wrote in Python. The idea brewed over dinner with my friend Josh -- creating a completely obviously trivial non-choose your own adventure game. The idea is that the templates for the adventures are suitably generic, and then the fillers convert them into specific genres (for fantasy, fill with THE WIZARD, etc.). The "non-choose" is because there is no actual choice -- there is no decision tree, and the user's input doesn't actually have any effect. The substitutions are made more obvious with the use of the termcolor library, and converting them to all uppercase. I couldn't get blinking text to work, otherwise I probably would have turned that on too. Yes, it's a bit of a joke, but I threw it up on github and put it under the GPLv3 anyway. I got a laugh out of it, and it took me a total of maybe 30 minutes to get completely working.

Dependencies

Python 2.6 (may work under other versions of Python 2.x)

Install

Run the script choose.py for easy access to pre-made scripts, or run basketball.py for the full interface, and to make new filler files. Template files currently need to be made by hand (this isn't terribly hard). See the README for more details.
